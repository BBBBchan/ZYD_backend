# ZYD_backend
This is a backend project for wxapp Zhongyidian.

## Database Init

> 1.python manage.py db init

> 2.python manage.py db.migrate

Enter shell:

> 3.python manage.py shell

> 4.db.create_all()

Create Roles:

> 5.create_roles()

Record Log:

> Create file /log/app.log at root path of project
